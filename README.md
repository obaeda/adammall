# موقع تنزيل التطبيق (GitHub Pages)

هذه صفحة ثابتة جاهزة للنشر على GitHub Pages. تعرض زرين لتحميل التطبيق (آيفون والأندرويد).

ملفات موجودة:
- `index.html`
- `styles.css`

خطوات سريعة للنشر على GitHub Pages:

1. أنشئ مستودع جديد على GitHub (مثلاً `my-app-landing`).
2. انسخ الملفات الموجودة هنا إلى المستودع أو ادفعها عبر `git` إلى الفرع `main` أو `gh-pages`.
3. في صفحة المستودع على GitHub: اختر **Settings → Pages**، وحدد مصدر النشر `main` branch و`/ (root)` ثم اضغط Save.
4. بعد دقائق قليلة سيصبح موقعك متاحًا على `https://<your-username>.github.io/<repo-name>/`.

تعديل روابط التنزيل:
- افتح `index.html` وابحث عن الروابط `https://example.com/ios` و`https://example.com/android` واستبدلهما بروابط متجر App Store وGoogle Play الخاصة بتطبيقك.

هل تريد أن أدفع هذا كـ repository إلى GitHub وأقوم بتمكين Pages لك؟ إذا نعم، أعطني صلاحية (وإلا سأوجهك بالأوامر اللازمة).
